title: Hybrid Search
description: Learn how to perform hybrid search in Orama Cloud.

Hybrid search is the combination of both vector and full-text search. It combines the precision of full-text search with the semantic understanding of vector search using weighted scoring to balance exact keyword matching with semantic relevance, making it ideal for most production applications despite being more complex to tune and computationally expensive than using either method alone.

## Hybrid Search APIs

The hybrid search APIs in Orama Cloud are basically the same as the full-text and vector search APIs. They allow you to perform hybrid search on your indexed documents by just setting `mode` to `hybrid`.

<Tabs items={['JavaScript', 'Python', 'Rust']}>
  <Tab value="Javascript">
  ```typescript
  import { OramaCloud } from '@orama/core'

  const orama = new OramaCloud({
    projectId: '<your_project_id>',
    apiKey: '<your_private_api_key>',
  })

  const searchResults = await orama.search({
    term: 'The quick brown fox jumps over the lazy dog',
    // [!code highlight]
    mode: 'hybrid' // [!code highlight]
  })
  ```
  </Tab>
  <Tab value="Python">
  ```python
  import asyncio
  from oramacore_client import OramaCloud

  async def main():
      orama = OramaCloud(
          project_id="<your_project_id>",
          api_key="<your_private_api_key>",
      )
      search_results = await orama.search(
          term="The quick brown fox jumps over the lazy dog",
          # [!code highlight]
          mode="hybrid" # [!code highlight]
      )
      print(search_results)

  if __name__ == "__main__":
      asyncio.run(main())
  ```
  </Tab>
  <Tab value="Rust">
  ```rust
  use oramacore_client::{
      OramaCloud,
      types::{CloudSearchParams, SearchMode},
  };

  #[tokio::main]
  async fn main() {
      let orama = OramaCloud::new("<your_project_id>", "<your_private_api_key>");

      let search_params = CloudSearchParams::new(
          "The quick brown fox jumps over the lazy dog",
      // [!code highlight]
      ).with_mode(SearchMode::Hybrid); // [!code highlight]

      let search_results = orama
          .search(&search_params)
          .await
          .unwrap();

      println!("{:?}", search_results);
  }
  ```
  </Tab>
</Tabs>

## Filtering

You can use all the filtering options available in the full-text and vector search APIs even when performing hybrid search. The only difference is that you have to set `mode` to `hybrid` when performing vector search:

<Tabs items={['JavaScript', 'Python', 'Rust']}>
  <Tab value="Javascript">
  ```typescript
  import { OramaCloud } from '@orama/core'

  const orama = new OramaCloud({
    projectId: '<your_project_id>',
    apiKey: '<your_private_api_key>',
  })

  const searchResults = await orama.search({
    term: 'A pair of stylish and elegant shoes for a grand gala event',
    // [!code highlight]
    mode: 'hybrid', // [!code highlight]
    where: {
      price: {
        lte: 100
      }
    },
    sortBy: {
      property: 'price',
      order: 'DESC'
    }
  })
  ```
  </Tab>
  <Tab value="Python">
  ```python
  import asyncio
  from oramacore_client import OramaCloud

  async def main():
      orama = OramaCloud(
          project_id="<your_project_id>",
          api_key="<your_private_api_key>",
      )
      search_results = await orama.search(
          term="A pair of stylish and elegant shoes for a grand gala event",
          # [!code highlight]
          mode='hybrid', # [!code highlight]
          where={
              "price": {"lte": 100}
          },
          exact=True,
          sort_by={
              "property": "price",
              "order": "DESC"
          }
      )
      print(search_results)

  if __name__ == "__main__":
      asyncio.run(main())
  ```
  </Tab>
  <Tab value="Rust">
  ```rust
  use oramacore_client::{
      OramaCloud,
      types::{SearchOptions, SortBy, Order, SearchMode},
  };
  use std::collections::HashMap;

  #[tokio::main]
  async fn main() {
      let orama = OramaCloud::new("<your_project_id>", "<your_private_api_key>");

      let mut where_filter = HashMap::new();
      where_filter.insert(
          "price".to_string(),
          serde_json::json!({ "lte": 100 })
      );

      let search_options = SearchOptions {
          term: "A pair of stylish and elegant shoes for a grand gala event".to_string(),
          // [!code highlight]
          mode: Some(SearchMode::Hybrid), // [!code highlight]
          where_filter: Some(where_filter),
          exact: Some(true),
          sort_by: Some(SortBy {
              property: "price".to_string(),
              order: Order::Desc,
          }),
          ..Default::default()
      };

      let search_results = orama
          .search(search_options)
          .await
          .unwrap();

      println!("{:?}", search_results);
  }
  ```
  </Tab>
