title: Vector Search
description: Learn how to perform vector search in Orama Cloud.

At indexing-time, Orama Cloud automatically generates a vector representation of each document. This vector is used to perform vector search, which is a powerful technique for finding documents that are similar to a given query.

Orama also transforms your input query into a vector representation at query-time. This vector is then used to find the most similar documents to the query.

No configuration is required for vector search in Orama Cloud. Simply index your documents and perform vector search using the provided API.

## Vector Search APIs

The vector search APIs in Orama Cloud are basically the same as the full-text search APIs. They allow you to perform vector search on your indexed documents by just setting `mode` to `vector`.

Optionally, you can also specify a `similarity` property, which is a number between 0 and 1 that determines how similar the documents must be to the query. A higher value means that the documents must be more similar to the query.

<Tabs items={['JavaScript', 'Python', 'Rust']}>
  <Tab value="Javascript">
  ```typescript
  import { OramaCloud } from '@orama/core'

  const orama = new OramaCloud({
    projectId: '<your_project_id>',
    apiKey: '<your_private_api_key>',
  })

  const searchResults = await orama.search({
    term: 'The quick brown fox jumps over the lazy dog',
    // [!code highlight]
    mode: 'vector',
    similarity: 0.8 // [!code highlight]
  })
  ```
  </Tab>
  <Tab value="Python">
  ```python
  import asyncio
  from oramacore_client import OramaCloud

  async def main():
      orama = OramaCloud(
          project_id="<your_project_id>",
          api_key="<your_private_api_key>",
      )
      search_results = await orama.search(
          term="The quick brown fox jumps over the lazy dog",
          # [!code highlight]
          mode="vector",
          similarity=0.8 # [!code highlight]
      )
      print(search_results)

  if __name__ == "__main__":
      asyncio.run(main())
  ```
  </Tab>
  <Tab value="Rust">
  ```rust
  use oramacore_client::{
      OramaCloud,
      types::{CloudSearchParams, SearchMode},
  };

  #[tokio::main]
  async fn main() {
      let orama = OramaCloud::new("<your_project_id>", "<your_private_api_key>");

      let search_params = CloudSearchParams::new(
          "The quick brown fox jumps over the lazy dog",
      )
      .with_mode(SearchMode::Vector) // [!code highlight]
      .with_threshold(0.8); // [!code highlight]

      let search_results = orama
          .search(&search_params)
          .await
          .unwrap();

      println!("{:?}", search_results);
  }
  ```

## Filtering

You can use all the filtering options available in the full-text search APIs even when performing vector search. The only difference is that you have to set `mode` to `vector` when performing vector search:

<Tabs items={['JavaScript', 'Python', 'Rust']}>
  <Tab value="Javascript">
  ```typescript
  import { OramaCloud } from '@orama/core'

  const orama = new OramaCloud({
    projectId: '<your_project_id>',
    apiKey: '<your_private_api_key>',
  })

  const searchResults = await orama.search({
    term: 'A pair of stylish and elegant shoes for a grand gala event',
    // [!code highlight]
    mode: 'vector',
    similarity: 0.8 // [!code highlight]
    where: {
      price: {
        lte: 100
      }
    },
    sortBy: {
      property: 'price',
      order: 'DESC'
    }
  })
  ```
  </Tab>
  <Tab value="Python">
  ```python
  import asyncio
  from oramacore_client import OramaCloud

  async def main():
      orama = OramaCloud(
          project_id="<your_project_id>",
          api_key="<your_private_api_key>",
      )
      search_results = await orama.search(
          term="A pair of stylish and elegant shoes for a grand gala event",
          # [!code highlight]
          mode='vector',
          similarity=0.8 # [!code highlight]
          where={
              "price": {"lte": 100}
          },
          exact=True,
          sort_by={
              "property": "price",
              "order": "DESC"
          }
      )
      print(search_results)

  if __name__ == "__main__":
      asyncio.run(main())
  ```
  </Tab>
  <Tab value="Rust">
  ```rust
  use oramacore_client::{
      OramaCloud,
      types::{SearchOptions, SortBy, Order, SearchMode},
  };
  use std::collections::HashMap;

  #[tokio::main]
  async fn main() {
      let orama = OramaCloud::new("<your_project_id>", "<your_private_api_key>");

      let mut where_filter = HashMap::new();
      where_filter.insert(
          "price".to_string(),
          serde_json::json!({ "lte": 100 })
      );

      let search_options = SearchOptions {
          term: "A pair of stylish and elegant shoes for a grand gala event".to_string(),
          mode: Some(SearchMode::Vector), // [!code highlight]
          threshold: Some(0.8), // [!code highlight]
          where_filter: Some(where_filter),
          exact: Some(true),
          sort_by: Some(SortBy {
              property: "price".to_string(),
              order: Order::Desc,
          }),
          ..Default::default()
      };

      let search_results = orama
          .search(search_options)
          .await
          .unwrap();

      println!("{:?}", search_results);
  }
  ```
