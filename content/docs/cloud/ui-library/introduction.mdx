## Get started

| Prerequisites: Orama UI is designed to work with Orama Cloud projects. You'll need an active account and project set up on the Orama Cloud platform before using these components. 

Install the package:

```bash
npm install @orama/ui
# or
yarn add @orama/ui
# or
```
